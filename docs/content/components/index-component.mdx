---
title: Index component
---

The `Index` component gives you an unnumbered list containing
the children of the current section.

<Screenshot src="/components/index-component.png" alt="Screenshot of the index component" />

Options:

* `root`: the path to the root to display (optional)
* `depth`: the maximum depth to display (optional)

Example:

```html
<Index depth="1" />
```
